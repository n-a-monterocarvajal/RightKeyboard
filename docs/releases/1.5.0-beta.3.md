# RightKeyboard 1.5.0-beta.3

Tercera beta pública de RightKeyboard 1.5. Corrige problemas de interacción y acabado detectados durante la prueba física de beta 2.

## Correcciones

- Es posible escribir y editar el alias sin que el feedback de Raw Input cambie selección o foco.
- La lista prioriza teclados conectados y configurados; los ignorados quedan al final.
- Desaparece el elemento vacío rojizo del selector de distribuciones.
- Los diálogos de confirmación usan botones y superficies redondeadas coherentes con Fluent.
- El selector automático solicita activación, pasa al frente y enfoca el nombre del teclado.

## Validación

- Compilación Release sin errores ni advertencias.
- 83 de 83 pruebas automatizadas correctas.
- Queda pendiente validar foco, temas y DPI en varios equipos físicos.

## Pendiente antes de la estable

Se realizará una revisión integral de los textos de interfaz para acortar formulaciones excesivamente descriptivas o técnicas, conservando accesibilidad y capacidad de diagnóstico.
