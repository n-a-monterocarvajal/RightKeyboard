# RightKeyboard 1.4.0

Primera versión modernizada de RightKeyboard, validada en una máquina host con varios teclados.

## Cambios principales

- Migración de .NET Framework a .NET 10 LTS.
- Sustitución del formulario principal oculto por una ventana de mensajes liviana.
- Menú del área de notificación traducido y limitado a **Limpiar preferencias** y **Salir**.
- Lectura completa de eventos Raw Input para ignorar liberaciones, modificadores y eventos falsos al configurar dispositivos.
- Agrupación de funciones HID mediante el identificador de contenedor de Windows.
- Cambio de distribución dirigido solamente a la ventana activa.
- Corrección del identificador de hilo truncado en la implementación heredada.
- Documentación, selector, mensajes y registro de cambios en español.
- Doce pruebas automatizadas.

## Descargas

- `RightKeyboard-1.4.0-win-x64-self-contained.zip`: versión recomendada; no requiere instalar .NET por separado.
- `RightKeyboard-1.4.0-win-x64-framework-dependent.zip`: descarga pequeña; requiere .NET 10 Desktop Runtime x64.
- `RightKeyboard-1.4.0-SHA256.txt`: sumas de comprobación de ambos paquetes.

Las preferencias se almacenan en `%LOCALAPPDATA%\RightKeyboard\config.txt`.

La hoja de ruta para identificación visible, persistencia mejorada, exclusión de periféricos compuestos y selector jerárquico está documentada en `ROADMAP.md`.
