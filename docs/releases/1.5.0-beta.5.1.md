# RightKeyboard 1.5.0 beta 5.1

Revisión puntual de beta 5 para corregir una regresión crítica sin adelantar el conjunto visual previsto para beta 6.

## Corrección

- Configuración ya no se cierra al interactuar por primera vez con «Nombre para este teclado».
- El aviso animado mantiene estado explícito y solo detiene su `Storyboard` después de haberlo iniciado.
- La animación puede reiniciarse de forma segura con cada edición del alias.

No incluye todavía las mejoras visuales y de microcopia anotadas para beta 6.
